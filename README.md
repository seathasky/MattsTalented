# Talented

![main](https://i.imgur.com/xSYv3kW.png)

Talented keeps talent tracking simple and provides a minimal look to the Talent system.

## Features

**Track your talents on screen:**

*   Shows your currently active saved talent load-out in a clean, minimal bar
*   Hover-reveal header, One-click **Change** button opens the Talent window
*   Lightweight style controls: bar width, background alpha, and font color/style
*   Audio/visual feedback when talents have changed.
